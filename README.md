# Super-Trunfo-Python

Esse é o meu primeiro projeto pessoal que criei para exercitar meus conhecimentos em Python. Nele Será executado um programa que irá rodar o jogo 'Super Trunfo' no terminal do computador.
O programa irá rodar em pegando os dados de um arquivo .json com várias 'cartas' com atributos diferentes.

# Regra do Super Trunfo
* Será sorteado uma carta aleatória para o jogador e o oponente, cada carta contem 3 atributos de valor de 1 até 10.
* O jogador irá selecionar um desses atributos e as cartas serão comparadas com base nesses atributos, o que tiver o maior valor irá ganhar aquela rodada.
* Será sorteado uma nova carta para o jogador e o adversário e o processo se repete.
* Quando o jogador ganhar 5 rodadas seguidas o jogo é encerrado.	
* Caso o jogador perca 1 rodada, o jogo se encerra e a vitória é do oponente.
